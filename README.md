# Getting Started

1. `$ yarn install`
2. `$ yarn dev`

# Usage

- Once `$ yarn dev` is executed you can see the app in the browser at [http://localhost:3000](http://localhost:3000)
- Or you can see the app running in [https://css-for-js.vercel.app](https://css-for-js.vercel.app)
